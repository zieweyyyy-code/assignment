常见编程错误示例

错误1：未关闭的文件流（资源泄漏）

错误代码（Python）：

file = open('data.txt', 'r')
content = file.read()
# file.close() 被遗忘

解决方案：

with open('data.txt', 'r') as file:
    content = file.read()
# 文件会自动关闭

错误2：数组越界访问

错误代码（Java）：

int[] numbers = {1, 2, 3};
System.out.println(numbers[3]); // 报错！

解决方案：

if (index < numbers.length) {
    System.out.println(numbers[index]);
}

错误3：JavaScript 循环闭包问题

错误代码：

for (var i = 0; i < buttons.length; i++) {
    buttons[i].onclick = function() {
        console.log(i); // 总是输出 buttons.length
    }
}

解决方案：使用 let 或闭包

for (let i = 0; i < buttons.length; i++) {
    buttons[i].onclick = function() {
        console.log(i); // 正确输出
    }
}
