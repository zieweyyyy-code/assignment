常见编程错误示例

错误1：未关闭的文件流（资源泄漏）

错误代码（Python）：

file = open('data.txt', 'r')
content = file.read()
# ...其他代码...
# file.close()  被遗忘的关闭操作

解决方案：

使用with语句自动管理文件资源： 
with open('data.txt', 'r') as file:
    content = file.read()
# 文件会在代码块结束后自动关闭
或者在finally块中确保关闭文件： with open('data.txt', 'r') as file: content = file.read() # 文件会在离开with块时自动关闭 或者确保手动调用file.close()。

错误2：数组越界访问（索引超出范围）

错误代码（Java）：

int[] numbers = {1, 2, 3};
System.out.println(numbers[3]); // ArrayIndexOutOfBoundsException
// Java数组索引从0开始，有效索引是0-2，而numbers[3]会抛出异常

解决方案：

方法1：检查数组长度后再访问

if (numbers.length > index) {
    System.out.println(numbers[index]);
} else {
    System.out.println("索引超出范围");
}

方法2：使用增强for循环避免索引问题（当不需要索引时）

for (int num : numbers) {
    System.out.println(num);
}

方法3：使用Optional处理可能不存在的元素（Java8+）

Optional.ofNullable(index < numbers.length ? numbers[index] : null)
         .ifPresent(System.out::println);

注意：不同语言处理方式不同，但核心思路都是要先检查边界。
